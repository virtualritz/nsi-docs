The Rust API
============

The `nsi` crate provides Rust wrappers for the |nsi| API.
These are based on the low-level wrapper crate `nsi-sys` that contains
autogenerated bindings on top of :doc:`nsi.h`.

Creating a Context
------------------

The core of these wrappers is the ``Context`` struct. Its construction
triggers dynamic linking with the renderer.

.. code-block:: rust
    :linenos:

    let ctx = nsi::Context::new(None)?
