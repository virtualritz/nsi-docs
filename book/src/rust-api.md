# The Rust API

The [nsi]{.title-ref} crate provides Rust wrappers for the ɴsɪ API. These are based on the low-level wrapper crate [nsi-sys]{.title-ref} that contains autogenerated bindings on top of \[nsi.h\](nsi.h.md).

## Creating a Context

The core of these wrappers is the `Context` struct. Its construction triggers dynamic linking with the renderer.

```{.rust linenos=""}
let ctx = nsi::Context::new(None)?
```
